This is a budget helper built by me and for me.

ONLY INPUT TRANSACTIONS FROM THE SAME MONTH!

I will add the feature to handle more than one month, but for now one month of transactions seems reasonable amount.

If you'd like to use it, format the input like this:

01/01/2000,Me,Income,+5000
01/01/2000,Wife,Income,+5000
01/01/2020,Me,Needs,-3000
01/01/2000,Me,Needs,-500
01/01/2000,Me,Wants,-30
01/31/2000,Wife,Wants,-100
